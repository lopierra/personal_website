My process (a work in progress)

  1. Create Github repo
  2. New R project from Github repo
  3. Install {blogdown} & {hugo}
devtools::install_github("rstudio/blogdown")

blogdown::install_hugo()
  1. Edit .gitignore
.Rproj.user
.Rhistory
.RData
.Ruserdata
blogdown
Thumbs.db # if a Mac user, .DS_Store instead  
public/ # if using Netlify
  1. Create example site with Academic theme
blogdown::new_site(theme = "gcushen/hugo-academic", 
                   theme_example = TRUE)

  1. Modify config.toml to change title & copyright

  2. Personalize content files

  • change content > authors > rename folder “admin” to “pierrette”
  • then change user to display in content > home > about.md
  • fill in or comment out information in config > _default > params.toml
  1. Remove a section (eg. “Demo”, “hero”) by going to content > home > demo.md and changing active = false. Delete navigation link in config > _default > menus.toml
